Nexus Calls API

API HTTP + Socket.IO para chamadas 1v1, voz em grupo, video em grupo, compartilhamento de tela, eventos de midia e sinalizacao WebRTC.

Base URL

https://call.ironvalecraft.shop

Rotas HTTP

GET /healthGET /config/publicGET /roomsPOST /roomsGET /rooms/:roomIdPATCH /rooms/:roomIdDELETE /rooms/:roomId

Eventos Socket.IO

join-room, leave-room, webrtc-offer, webrtc-answer, webrtc-ice-candidate, media-state, screen-share-start, screen-share-stop, voice-activity, room-message, reaction, participant-joined, participant-left

Exemplo frontend

import { io } from 'socket.io-client';

const socket = io('https://call.ironvalecraft.shop', { path: '/socket.io', transports: ['websocket', 'polling'] });

  roomId: 'chat_123',
  userId: 'user_123',
  displayName: 'Rafael',
  media: { audio: true, video: true, screen: false }
});
